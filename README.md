# CoupleConnect-v2.0
AI-powered relationship app with chat, partner connect, love score, and smart date planning features 💕
AI-powered relationship platform designed to help couples connect, communicate, and grow together ❤️

🚀 Live Features

✨ AI Relationship Advisor
Get smart emotional advice powered by AI (Bangla + English)

💬 Couple Chat System
Private chat between partners

💞 Partner Connect
Send & accept invites to connect with your partner

❤️ Love Compatibility Score
Analyze relationship strength using AI insights

📅 Smart Date Planner
Generate perfect romantic date plans

📍 Dating Location Finder
Find best places for dates based on city, mood, and budget

🤖 AI Virtual Partner
Chat with an AI companion (emotional + fun)

🎮 Love Games & Quiz
Fun couple quizzes + truth or dare

💌 Love Letter Generator
Generate beautiful personalized love letters

🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
Database: SQLite
AI: Groq / OpenAI-compatible API
Email Service: EmailJS
⚙️ Installation
git clone https://github.com/YOUR_USERNAME/couple-connect-ai.git
cd couple-connect-ai
pip install -r requirements.txt
🔐 Setup API Key

Create a file:

.streamlit/secrets.toml

Add your API key:

GROQ_API_KEY = "your_api_key_here"
▶️ Run Locally
streamlit run main.py
🌍 Deployment

Deploy easily using Streamlit Cloud:

Push project to GitHub
Go to Streamlit Cloud
Connect repo
Add secrets (API key)
Deploy 🚀
